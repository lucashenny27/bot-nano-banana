# bot-nano-banana
Bot que interactúa con nano banana y genera contenido de video para Instagram.

## Requisitos
- Python 3.10+
- Playwright
- Google Gemini API Key (configurada en el .env)

## Configuración para colaborar con Antigravity

Para que otro desarrollador pueda abrir este proyecto en Antigravity y continuar trabajando sin problemas:

1. **Clonar el repositorio**: `git clone https://github.com/lucashenny27/bot-nano-banana.git`
2. **Dependencias**: Ejecutar `pip install -r requirements.txt`
3. **Variables de Entorno**: Crea un archivo `.env` basado en las necesidades del proyecto.
4. **Archivo de Tareas**: Este proyecto incluye un archivo `task.md` que Antigravity lee automáticamente para entender el progreso actual.
